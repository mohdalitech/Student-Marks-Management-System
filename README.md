# Student Marks Management System

## Description

The **Student Marks Management System** is a beginner-friendly Python console application that stores and manages academic records for five students. The program collects student details, records marks in three subjects, calculates total marks and percentage, and determines whether each student has passed or failed based on subject-wise passing criteria.

This project was built to strengthen my understanding of Python fundamentals before learning loops and functions.

## Features

* Stores details of **5 students**
* Accepts student names and roll numbers
* Records marks in:

  * English
  * Mathematics
  * Science
* Calculates:

  * Total marks
  * Percentage
* Determines pass/fail status using subject-wise conditions
* Displays a report for every student
* Uses formatted output for better readability


## Python Concepts Used

* Variables

* Strings
* Lists
* User Input
* Arithmetic Operators
* Conditional Statements (`if-else`)
* Formatted Strings (f-strings)
